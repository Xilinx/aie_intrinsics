Complex anti-symmetric multiply-accumulate intrinsic function with center-tap optimization with x buffer conjugation with pre-sub from x input buffer .

acc0 += z00*(cnj(x00) - cnj(y00)) + z01*(cnj(x01) - cnj(y01)) + z02*(cnj(x02) - cnj(y02)) + z03*cnj(x03)
acc1 += z10*(cnj(x10) - cnj(y10)) + z11*(cnj(x11) - cnj(y11)) + z12*(cnj(x12) - cnj(y12)) + z13*cnj(x13)
acc2 += z20*(cnj(x20) - cnj(y20)) + z21*(cnj(x21) - cnj(y21)) + z22*(cnj(x22) - cnj(y22)) + z23*cnj(x23)
acc3 += z30*(cnj(x30) - cnj(y30)) + z31*(cnj(x31) - cnj(y31)) + z32*(cnj(x32) - cnj(y32)) + z33*cnj(x33)
